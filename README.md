# .dotfiles

My personal dotfiles to share between OS installation.

# Install

1. Clone this repository

```bash
cd $HOME
git clone https://github.com/4ngelf/.dotfiles.git
```

2. Setup .dotfiles

```bash
cd $HOME/.dotfiles
bash -c ./setup
```
